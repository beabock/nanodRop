# nanodRop 0.1.0

* Initial release of nanodRop package
* Added `read_nanodrop()` function for importing NanoDrop spectral data
* Added `parse_waves()` function for analyzing wavelength differences
* Added example datasets `ds` and `ds1`
* Comprehensive test suite with testthat
* Package documentation with pkgdown
* Vignettes for basic data import and advanced analysis

## Features

- Easy import of NanoDrop One TSV files
- Automatic data cleaning and validation
- Tidy data output compatible with tidyverse
- Flexible wavelength analysis capabilities
- Multiple sample support
- Integration with ggplot2 for visualization
